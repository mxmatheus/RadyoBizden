# Radyo Türkiye - Glassmorphism

Next.js 14 (App Router) tabanlı, tam glassmorphism tasarımlı canlı Türk radyo platformu. 

## Proje Hakkında
Bu proje, modern web standartlarını (Framer Motion, Zustand, Tailwind CSS) kullanarak Aurora/kuzey ışıkları arka planı, glassmorphism UI elementleri ve kesintisiz radyo dinleme deneyimi sunmayı hedefler.
